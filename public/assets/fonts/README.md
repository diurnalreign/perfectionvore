# Fuentes

Aquí vive el font **True Lies** usado por el wordmark `PERFECTIONVORE` del Hero.
El `@font-face` está definido en `src/index.css`.

Archivos:

- `true-lies.woff2` — usado en web (formato principal, ~37 KB).
- `true-lies.ttf` — original, como respaldo.
- `LICENSE-True-Lies.txt` — licencia original del font.

Si el font no cargara, el wordmark cae automáticamente en **Oswald**, así que el
sitio nunca se rompe.
