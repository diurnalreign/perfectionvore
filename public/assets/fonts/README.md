# Fuentes

Coloca aquí el archivo del font **True Lies** usado por el wordmark
`PERFECTIONVORE` del Hero. El `@font-face` está definido en `src/index.css`.

Nombres de archivo esperados (uno u otro; `.woff2` es lo ideal por peso):

- `true-lies.woff2`
- `true-lies.ttf`

Mientras el archivo no exista, el wordmark cae automáticamente en **Oswald**,
así que el sitio nunca se rompe.

> ⚠️ Licencia: "True Lies" es *free for personal use*. Incrustarlo en un sitio
> público es una zona gris; úsalo bajo tu criterio o consigue una licencia si
> el proyecto crece. Por eso el archivo **no** se versiona en el repo.
